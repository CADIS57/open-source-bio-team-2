권민준: 조장, 페이지 틀, 버튼 클릭 기능, 장식 크기 변경 기능, 장식 강조 기능, 완성 표현, 브랜치 생성-변경, 버전, readme 작성
김가희: 장식 움직임 기능, prototype
김다연: 장식 색깔 변경 기능,장식 추가 기능, 상호작용 기능, 장식 모양 변경 기능

## 실행 방법
```bash
git clone https://github.com/CADIS57/open-source-bio-team-2
cd open-source-bio-team-2-main
html version_1_1
```
